### 회귀 : 단순선형회귀 / 다중선형회귀 / 다항 회귀
#### 단순선형회귀 : 단일특징, 단일 반응 변수 (y = ax)
#### 다중선형회귀 : 다수특징, 단일 반응 변수 (데이터 특징 증가, 차원증가(평면증가)) 행렬연산 (y = ax1+bx2+cx3+...) # N차원 공간
#### 다항 회귀 : 설명 변수와 반응 변수의 비선형 관계(x^n차식, 데이터 특징 증가, 곡선 모델) (y = ax^1+bx^2+cx^3+...) : 2차원 공간
